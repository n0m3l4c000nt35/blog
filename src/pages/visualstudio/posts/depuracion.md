# Depuración

| Atajo        | Descripción                      |
| ------------ | -------------------------------- |
| `F5`         | ejecutar con depuración          |
| `F9`         | breakpoint                       |
| `F10`        | paso a paso (sin entrar)         |
| `F11`        | paso a paso (entrando al método) |
| `Shift + F5` | detener                          |
